Time Travel Quiz

Chào mừng bạn đến với ứng dụng quản lý câu hỏi lịch sử "Time Travel Quiz".

Chức năng chính:

  • Thêm câu hỏi thủ công - Tạo và thêm câu hỏi trắc nghiệm vào hệ thống.
  • Xem danh sách câu hỏi - Duyệt qua tất cả các câu hỏi trong cơ sở dữ liệu.
  • Quản lý thời kỳ - Thêm, sửa hoặc xóa thời kỳ.
  • Upload câu hỏi từ file CSV - Thêm nhiều câu hỏi cùng lúc bằng file CSV.
  • Tạo câu hỏi tự động - Sử dụng AI để tự động tạo câu hỏi cho một thời kỳ.
  • API: Xem câu hỏi dưới dạng JSON - Truy cập dữ liệu câu hỏi dưới dạng JSON.
  • API: Xem thời kỳ dưới dạng JSON - Truy cập dữ liệu thời kỳ dưới dạng JSON.

Hướng dẫn sử dụng:

  1. Chọn chức năng từ danh sách trên.
  2. Làm theo các hướng dẫn trên mỗi trang để thêm, xem hoặc quản lý dữ liệu.
  3. Sử dụng API để tích hợp dữ liệu vào ứng dụng khác.

Ghi chú: Vui lòng đảm bảo rằng dữ liệu thời kỳ đã được thêm trước khi tạo hoặc upload câu hỏi.
